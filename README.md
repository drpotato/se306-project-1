## Prerequisites
1. Ubuntu 14.04
2. ROS Indigo

## Get the code
`git clone https://github.com/drpotato/se306-project-1.git`

## Run the program
1. Navigate to the home directory  
`cd se306-project-1`
2. Build the project  
`./build.sh`
3. Run the project  
`./run.sh`

See [Controls](https://github.com/drpotato/se306-project-1/wiki/Controls) for instructions for use
